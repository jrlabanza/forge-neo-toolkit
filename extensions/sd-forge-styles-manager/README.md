# sd-forge-styles-manager
"Styles" tab: live search over styles.csv (name + prompt text), in-place edit / add / delete with an automatic timestamped backup to `_attic/styles-backups/` before every write (newest 20 kept), send any style to txt2img/img2img, and a sample image from your own outputs found via the Gallery cache. The native styles dropdown refreshes automatically after saving.
